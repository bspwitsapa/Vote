# Project Instructions: Student Voting System (BSPWIT)

## Branding & Visual Identity
- **Institution:** Bangsaphan Wittaya School (โรงเรียนบางสะพานวิทยา)
- **Primary Colors:** Green and Yellow (เขียว-เหลือง)
  - Green (Primary): #15803d
  - Yellow (Accent): #facc15
- **Theme:** Professional, educational, and modern.

## Technical Context
- **Frontend:** HTML5, CSS3 (Vanilla CSS), JavaScript (ES6+).
- **Backend/Database:** Firebase (Auth, Firestore).
- **Responsiveness:** Must support mobile devices (Responsive Design).

## Project Guidelines
- Use 'Sarabun' as the primary font for Thai text.
- Maintain a clean and user-friendly interface for students and teachers.
